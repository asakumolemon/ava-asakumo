# Asakumo 界面原型设计 v2.0

> 基于 `avalonia-chat-ui-design` 和 `avalonia-ui-prototype` Skills 优化

---

## 设计规范

### 配色方案

采用 **OpenAI 风格**配色，专业、科技感：

| 颜色角色 | 浅色主题 | 深色主题 | 用途 |
|----------|----------|----------|------|
| BrandPrimary | `#10A37F` | `#10A37F` | 发送消息背景、主按钮 |
| BrandSecondary | `#0D8A6A` | `#0D8A6A` | 悬停状态 |
| Surface | `#FFFFFF` | `#1A1A1A` | 页面背景 |
| SurfaceVariant | `#F3F4F6` | `#2D2D2D` | 卡片背景、接收消息 |
| TextPrimary | `#1F2937` | `#F3F4F6` | 主要文字 |
| TextSecondary | `#6B7280` | `#9CA3AF` | 次要文字、时间戳 |
| TextOnPrimary | `#FFFFFF` | `#FFFFFF` | 主色上的文字 |

### 组件规范

| 组件 | 规范 |
|------|------|
| 聊天气泡 | 圆角 16px，一侧尖角 4px，MaxWidth 280px |
| 会话卡片 | 圆角 12px，内边距 12px |
| 主按钮 | 圆角 25px，内边距 24x14 |
| 输入框容器 | 圆角 24px |
| 头像 | 圆形，48x48px |

### 聊天气泡设计规范

**发送消息 (用户消息，右侧)：**

```xml
<Border Background="#10A37F"
        CornerRadius="16,4,16,16"
        Padding="12,8"
        HorizontalAlignment="Right"
        MaxWidth="280">
  <StackPanel Spacing="4">
    <TextBlock Text="{Binding Content}" 
               TextWrapping="Wrap" 
               Foreground="White"/>
    <TextBlock Text="{Binding Timestamp}" 
               FontSize="10" 
               Foreground="#80FFFFFF"
               HorizontalAlignment="Right"/>
  </StackPanel>
</Border>
```

**接收消息 (AI 消息，左侧)：**

```xml
<StackPanel Orientation="Horizontal" Spacing="8">
  <!-- 头像 -->
  <Border Width="32" Height="32" CornerRadius="16">
    <Image Source="/Assets/ai-avatar.png"/>
  </Border>
  
  <!-- 消息气泡 -->
  <Border Background="{DynamicResource SystemControlBackgroundChromeMediumBrush}"
          CornerRadius="4,16,16,16"
          Padding="12,8"
          MaxWidth="280">
    <StackPanel Spacing="4">
      <TextBlock Text="{Binding Content}" TextWrapping="Wrap"/>
      <TextBlock Text="{Binding Timestamp}" 
                 FontSize="10" 
                 Foreground="{DynamicResource SystemControlForegroundBaseMediumBrush}"
                 HorizontalAlignment="Right"/>
    </StackPanel>
  </Border>
</StackPanel>
```

**关键设计点：**
| 属性 | 发送消息 | 接收消息 |
|------|----------|----------|
| 对齐 | Right | Left |
| 背景 | `#10A37F` | SurfaceVariant |
| 圆角 | 16,4,16,16 | 4,16,16,16 |
| 文字 | White | TextPrimary |
| MaxWidth | 280px | 280px |
| 头像 | 无 | 32x32 圆形 |
